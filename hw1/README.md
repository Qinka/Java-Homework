Hello
===

This app is a "Hello World" in Java.

## Usage

This app can say hello to world or the The name you typed in.

```
$ java Hello
Hello, world!
```

or

```
java Hello John
Hello, John!
java Hello John Tom
Hello, John, and Tom!
```

## Build

Just use `javac` to build:

```
javac Hello.java
```
